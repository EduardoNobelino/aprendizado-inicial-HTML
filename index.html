História dos computadores

Os computadores evoluíram muito desde seus primeiros modelos. No início do século XX, máquinas como o ENIAC eram gigantescos e usados apenas para cálculos complexos. Com o tempo, surgiram os microprocessadores, que permitiram o desenvolvimento de computadores pessoais.

Na década de 1980, empresas como IBM e Apple popularizaram os PCs. Hoje, temos dispositivos poderosos e portáteis que cabem no bolso, conectados à internet e com acesso a informações em tempo real.

    • Diferença entre Servers e Clients

    Os servers são computadores que fornecem serviços, como hospedagem de sites e armazenamento de dados. Eles processam solicitações dos clientes e enviam respostas. Já os clients são dispositivos que acessam esses serviços, como computadores, smartphones e tablets. A comunicação entre eles é feita através de protocolos, como o HTTP.

    Em resumo, Clients são dispositivos que consome a internet e Servers fornecem informações para os clients.

    • Tags Aprendidas

    • Títulos:
      1. h1 - tag utilizada para o Título principal da página.
      2. h2 - tag utilizada para o subtítulo do título(h1) da página.
      3. h3 - tag utilizada para o subtítulo do subtítulo(h2) da página.
      4. h4 ao h6 - tag utilizada para organizar conteúdos menos importantes ou seções mais internas.
    • Texto:
      1. p - tag utilizada para criar parágrafos de texto.
      2. strong - tag utilizada para destacar texto em negrito.
      3. i - tag utilizada para destacar texto em itálico.
      4. mark - tag utilizada para destacar uma parte do texto (Como um marca-texto).
      5. small - tag utilizada para textos com a fonte menor.
      6. u - tag utilizada para sublinhar um texto.
    • Listas:
      1. ul - tag utilizada para criar listas não ordenadas.
      2. ol - tag utilizada para criar listas ordenadas.
      3. li - tag utilizada para definir itens de lista.
    • Links:
      1. a - tag utilizada para criar links.
    • Outros:
      1. hr - tag utilizada para criar uma linha de separação.
      2. sub - tag utilizada para colocar texto menor e abaixo da linha.
      3. sup - tag utilizada para colocar texto menor e acima da linha.
      4. blockquote - tag utilizada para definir uma citação em bloco.
